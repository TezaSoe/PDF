This directory contains a vector graphics version of the PDFBinder logo
(SVG format), along with a 500x500 bitmap version and the original artwork
used to compose the logo.

All individual elements are released into the "public domain" by their
authors. This includes the files mentioned below:

  go-next.svg
  text-x-generic.svg 

The composition of the PDFBinder logo is done by Joern Schou-Rode, and the
result is licensed for any use directly related to the PDFBinder project:
